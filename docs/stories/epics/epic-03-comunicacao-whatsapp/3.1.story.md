## Story

**As a** desenvolvedor do Advezo,
**I want to** substituir a integração WhatsApp baseada em Chromium/Puppeteer pela biblioteca Baileys,
**so that** o worker seja estável em Railway sem consumo excessivo de memória e sem dependência de browser headless.

> ⚠️ **Bloqueante para development das waves 2–5 do Epic 3:** confirmação das 24h de estabilidade em staging (AC 3.1.4) desbloqueia as stories seguintes.
> ⛔ **Bloqueante para produção com cliente real:** gate separado (`pre_production_gate` no EPIC-03-EXECUTION.yaml) — 7 dias de uptime contínuo monitorado antes de conectar número de cliente.

### whatsapp_accounts (Migration 000010)

```sql
CREATE TABLE public.whatsapp_accounts (
  id               uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  workspace_id     uuid NOT NULL REFERENCES public.workspaces(id) ON DELETE CASCADE,
  account_id       text NOT NULL,  -- E.164, ex: 5511999998888
  status           text NOT NULL DEFAULT 'disconnected'
                   CHECK (status IN ('disconnected','connecting','connected','cb_paused')),
  cb_failure_count integer NOT NULL DEFAULT 0,
  cb_paused_at     timestamptz,    -- NULL = circuit fechado; NOT NULL = circuit aberto
  connected_at     timestamptz,
  created_at       timestamptz NOT NULL DEFAULT now(),
  updated_at       timestamptz NOT NULL DEFAULT now(),
  UNIQUE (workspace_id, account_id)
);

-- RLS
ALTER TABLE public.whatsapp_accounts ENABLE ROW LEVEL SECURITY;
CREATE POLICY workspace_isolation ON public.whatsapp_accounts
  USING (workspace_id = auth_workspace_id());
```

### Supabase Storage — Sessão Baileys

Caminho: `{workspace_id}/wpp/{account_id}/session.json`

```typescript
// src/session.ts
import { createClient } from '@supabase/supabase-js'

const BUCKET = 'wpp'

export async function saveSession(
  workspaceId: string,
  accountId: string,
  sessionData: unknown
): Promise<void> {
  const supabase = createClient(process.env.SUPABASE_URL!, process.env.SUPABASE_SERVICE_ROLE_KEY!)
  const path = `${workspaceId}/wpp/${accountId}/session.json`
  await supabase.storage.from(BUCKET).upload(path, JSON.stringify(sessionData), {
    upsert: true,
    contentType: 'application/json',
  })
}

export async function restoreSession(
  workspaceId: string,
  accountId: string
): Promise<unknown | null> {
  const supabase = createClient(process.env.SUPABASE_URL!, process.env.SUPABASE_SERVICE_ROLE_KEY!)
  const path = `${workspaceId}/wpp/${accountId}/session.json`
  const { data, error } = await supabase.storage.from(BUCKET).download(path)
  if (error || !data) return null
  return JSON.parse(await data.text())
}
```
