### normalizePhone — Contrato Exato

```typescript
// packages/utils/src/phone.ts
export function normalizePhone(phone: string): string {
  const digits = phone.replace(/\D/g, '')
  const with9 = digits.length === 10
    ? digits.slice(0, 2) + '9' + digits.slice(2)
    : digits
  return with9.startsWith('55') ? with9 : `55${with9}`
}
```

Casos de teste obrigatórios:
- `'(11) 9999-8888'` → `'5511999988888'`
- `'11 8888-7777'` (10 dígitos) → `'5511988887777'`
- `'+55 11 99999-8888'` → `'5511999998888'`
- `'5511999998888'` (já E.164) → `'5511999998888'`
