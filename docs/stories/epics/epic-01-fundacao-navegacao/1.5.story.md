## Story

**As a** gestor de tráfego,
**I want to** ter uma barra lateral com navegação de 7 itens, seletor de cliente e seletor de conta de anúncios,
**so that** eu possa alternar rapidamente entre clientes e contas sem perder o contexto da tela atual.

### Sidebar Anatomy (docs/ux-design-spec.md Section 2 — atualizado 2026-06-25)

```
┌─────────────────────────────────────────┐
│ [≡] Advezo                         [<]  │  ← header: logo + collapse toggle
├─────────────────────────────────────────┤
│ Workspace: Agência XYZ              [v] │  ← workspace display (não clicável ainda)
│ ┌─────────────────────────────────────┐ │
│ │ 🔍 Buscar cliente...               │ │  ← ClientSelector (Combobox)
│ └─────────────────────────────────────┘ │
│ Cliente Ativo                           │
│ ┌─────────────────────────────────────┐ │
│ │ 🔍 Conta de anúncios...            │ │  ← AdAccountSelector (condicional)
│ └─────────────────────────────────────┘ │
├─────────────────────────────────────────┤
│ LayoutDashboard   Dashboard             │  ← nav items (ordem final aprovada)
│ Users             Clientes              │
│ MessageCircle     WhatsApp              │  ← sub-nav no content area: Conexões|Conversas|Links
│ TrendingUp        Conversões            │

├─────────────────────────────────────────┤
│ Settings          Configurações         │  ← inclui sub-seção Equipe
├─────────────────────────────────────────┤
│ [avatar] Nome do Usuário           [v] │  ← avatar + dropdown
└─────────────────────────────────────────┘
```

**Configurações sub-seção Equipe:**

### TanStack Query — Fetch Clientes

```typescript
// apps/web/src/lib/queries/clients.ts
export async function fetchClients(): Promise<Client[]> {
  const res = await fetch('/api/clients')
  if (!res.ok) throw new Error('Failed to fetch clients')
  return res.json()
}
```

### Lista Final de Nav Items (decisão 2026-06-25)

Lista aprovada pelo usuário — **PRD e UX spec foram sincronizados** com esta lista:

1. Dashboard
2. Clientes
3. WhatsApp (sub-nav: Conexões | Conversas | Links Rastreáveis)
4. Conversões
5. Relatórios
6. Assistente IA
7. Configurações (inclui sub-seção Equipe — não é item de primeiro nível)

**"Rastreamento" como conceito:** fundia links rastreáveis + conversas + resultados. Na nova estrutura:
- Links rastreáveis → WhatsApp > Links Rastreáveis
- Conversas classificadas → WhatsApp > Conversas
- Dashboard de resultados/ROI → Conversões
